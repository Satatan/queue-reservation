# Queue-Reservation

Import Queue Reservation.postman_collection from /backend/app/api/postman

## There are 2 ways to run it.

### 1. run from docker-compose

```bash
docker-compose up --build
```

### 2. run from go

go to backend folder
```bash
cd .\backend\ 
```

run go
```bash
go run app/api/main.go
```
